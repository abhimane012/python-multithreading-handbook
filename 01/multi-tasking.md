# What is Multitasking?

Multitasking is the ability of a system to perform multiple tasks at the same time, improving efficiency and resource utilization.

## Types of Multitasking

1. **Process-Based Multitasking**  
    - Involves running multiple programs or processes independently.  
    - Each process operates in its own memory space.  
    - Commonly managed by the operating system to ensure isolation and stability.  
    - **Example**: Running a web browser, a text editor, and a music player simultaneously on your computer.

2. **Thread-Based Multitasking**  
    - Involves executing multiple threads within the same program.  
    - Threads share the same memory space, enabling faster communication.  
    - Often used in applications to perform concurrent operations efficiently.  
    - **Example**: A web server handling multiple client requests concurrently using threads.
