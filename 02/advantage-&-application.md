# Advantages

1. Better performance — uses multiple CPU cores for faster results.  
2. Lower response time — apps and websites respond more quickly.  
3. Multiple execution flows — lets a program run several tasks at the same time instead of only one.

# Common applications

- Video games (e.g., PUBG) — separate threads for rendering, physics, input, and networking.  
- Multimedia and graphics — parallel decoding and rendering.  
- Animations and smooth UIs — keep updates responsive.  
- Web servers — handle many requests at the same time.  
- General applications — background tasks, I/O handling, and responsiveness.
